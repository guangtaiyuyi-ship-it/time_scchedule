# 私のタイムスケジュール帳 📅

PythonとStreamlitで作成した、自分専用のパーソナル・スケジュール管理アプリケーションです。
1週間の時間の使い方を俯瞰し、日々のタスクやルーティンを視覚的に管理・分析することができます。

## ✨ 主な機能

- **基本テンプレート機能**: 毎週繰り返すルーティン（講義、アルバイトなど）を曜日ごとに登録。
- **個別予定の管理**: その日だけの特別な予定を柔軟に追加・編集・削除。
- **カテゴリとカラーのカスタマイズ**: タスクの種類ごとに色分けし、自分好みのカテゴリを自由に追加可能。
- **重複エラー防止機能**: 予定の時間が重なっている場合は自動でブロックし、入力ミスを防止。
- **ワークライフバランス分析**: 1週間の時間の使い方（学習、プライベートなど）をカテゴリごとに自動集計し、棒グラフで可視化。

## 🛠️ 使用技術 (Tech Stack)

- **Python** 3.13.9
- **Streamlit**: Webアプリケーションの構築（UI設計）
- **Pandas**: 予定データの処理・合体・集計
- **Plotly**: 円グラフや棒グラフによるデータの視覚化（ダッシュボード）
